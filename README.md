## Fonctionnalités

### L'utilisateur peut :
__En lançant le jeu__
- Naviguer dans le menu principal
- Naviguer dans les différents sous-menus ( option / jeu )
- Choisir le mode de jeu

__En jeu__
- Se déplacer
- Attaquer
- Ramasser une arme

__Pendant les affrontements__
- Subir des dégâts
- Mourir __-1 Score__
- Respawn
- Tuer quelqu'un __+3 Scores__
